# Terralith Inspired Biomes

![Terralith Inspired Biomes Preview](preview.png)

> A community Terralith-inspired world generation project for CastleForge's **WorldGen** category.

## Overview

This catalog entry links to the **TerralithInspiredBiomes** project.

It is intended for use with CastleForge world generation systems and focuses on dramatic terrain generation inspired by large cliffs, deep valleys, towering mountains, and biome-driven landscape variation.

- **Source:** https://github.com/RussDev7/CastleForge-TerralithInspiredBiomes

## Installation

1. Install the required CastleForge world generation framework in your CastleMiner Z setup.
2. Download the project from the linked repository or its Releases page.
3. Place the world generation files in the location expected by your world-gen system.
4. Launch the game and generate a new world using the included terrain or biome preset.
